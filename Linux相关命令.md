CPU占用率查看

> top  // 查看每个进程的CPU占用率，多核情况下可看到超过100%



内存占用率查看

> free -m



磁盘使用率查看

> df



网卡流量查看

> iftop  // 监控网卡的实时流量



测试主机之间网络的连通性

> ping



查看Linux中网络系统的状态信息，网络情况

> netstat



查看日志常用命令

> tail  // -n显示行号，后面行数
>
> head  // 反tail，前面行数
>
> cat  // 查看，tac倒序查看
>
> more / less  // 日志很多，分页打印



抓取网络数据包

> tcpdump -i + [网卡 ]     // 监听指定网卡
>
> tcpdump  -i + [网卡] + -nn ['网络协议 / 主机号 / 端口号']     // 监听指定网卡的指定协议或指定主机或指定端口
